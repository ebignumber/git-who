Install in 10 seconds

$ pip install git+https://github.com/trinarymage/git-who.git

$ cd your-project
$ git-who me          # Your personal profile
$ git-who health      # Letter grade for knowledge distribution
$ git-who hotspots    # Find your riskiest code
$ git-who diff        # Risk assessment for current changes
$ git-who map         # Interactive ownership treemap

22 features. Zero config.

Personal Profile

git-who me — See your impact, sole-expert files, and what happens if you leave.

Change Risk Assessment

git-who diff — Risk grade (A-F) per PR. Know before you merge.

Health Grade

git-who health — Letter grade for knowledge distribution. Instantly shareable.

Hotspot Detection

git-who hotspots — High churn + low bus factor = your riskiest code.

Interactive Treemap

git-who map — Zoomable, color-coded ownership visualization.

CODEOWNERS Generation

git-who codeowners — Auto-generate from actual expertise data.

Reviewer Suggestions

git-who review — Expertise-based reviewers, not random assignment.

GitHub Action

Run in CI with PR comments, bus factor gates, and risk reports.

Trend Analysis

git-who trend — Track bus factor over time with sparklines.
